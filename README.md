Interactive Power BI dashboard analyzing global economic and development indicators using World Bank data.

# 🌍 World Bank Analytics Dashboard

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** that analyzes global economic and development indicators using open datasets from the World Bank.

It helps in understanding how different countries perform across key metrics like GDP, population, inflation, unemployment, and life expectancy.

---

## 🎯 Objective

| Goal | Description |
|------|------------|
| 🌍 Global Analysis | Understand worldwide economic trends |
| 📊 Country Comparison | Compare countries on development indicators |
| 📈 KPI Tracking | Monitor GDP, inflation, population, etc. |
| 🔍 Insights | Identify top & bottom performing economies |
| 📊 Visualization | Build interactive dashboards for decision making |

---

| Excel / CSV | Data storage & preprocessing |
| World Bank Data | Primary dataset source |
| Python | Fetch data from World Bank Data |
| SQL | Queries and Data Enrichment |

---

## 📊 Key Metrics Used

| Metric | Description |
|--------|------------|
| GDP | Gross Domestic Product |
| Population | Total population per country |
| Inflation Rate | Economic stability indicator |
| Unemployment Rate | Labor market condition |
| Life Expectancy | Health & development indicator |
| GDP_Growth | Rate of Change of GDP |

---

## 🗂️ Dashboard Structure

### 🟦 1. Global Overview
- Worldwide summary of economic indicators  
- Key KPI cards (GDP, population, inflation)

---

### 🟩 2. Economic Performance
- Country-wise GDP comparison  
- Top 10 & Bottom 10 GDP Countries  
- Trend analysis over years 

---

### 🟨 3. Population & Health Insights
- Population distribution by country
- Life Expectancy distribution by country   
- Growth trends of population and life expectancy 

---

### 🟧 4. Inflation & Unemployment Insights
- Inflation comparison by country
- Unemloyment comparison by country 
- Inflation & unemployment trends
